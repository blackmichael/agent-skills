# agent-skills
Collection of useful agent skills.

## Recommended Installation

Use [npx skills](https://skills.sh/) to install skills from this repository.

```
npx skills add blackmichael/agent-skills --skill agentic-code-reasoning --agent claude-code codex
```

```
npx skills add blackmichael/agent-skills --skill design-doc-review --agent claude-code codex
```
